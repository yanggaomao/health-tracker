# Blood Glucose Ranges

Use these ranges as logging and visualization references, not as diagnosis. Prefer the user's clinician-provided targets when available.

## Unit Conversion

- mmol/L to mg/dL: multiply by 18.
- mg/dL to mmol/L: divide by 18.

## General Adult Reference Ranges

These ranges describe common lab-screening interpretations for nonpregnant adults:

| Timing | Normal / expected | Elevated concern |
| --- | --- | --- |
| Fasting | <100 mg/dL / <5.6 mmol/L | 100-125 mg/dL / 5.6-6.9 mmol/L can be prediabetes range in lab testing; >=126 mg/dL / >=7.0 mmol/L on lab testing can be diabetes range |
| 2 hours after glucose challenge or meal context | <140 mg/dL / <7.8 mmol/L | 140-199 mg/dL / 7.8-11.0 mmol/L can be prediabetes range in lab testing; >=200 mg/dL / >=11.1 mmol/L can be diabetes range |
| Low glucose threshold | <70 mg/dL / <3.9 mmol/L | Treat as possible hypoglycemia, especially with symptoms |

Home finger-stick readings are not the same as diagnostic lab venous plasma testing. Avoid diagnosing from a single home reading.

## Diabetes-Management Target Ranges

For many nonpregnant adults with diabetes, common self-monitoring targets are:

| Timing | Common target |
| --- | --- |
| Before meals / preprandial | 80-130 mg/dL / 4.4-7.2 mmol/L |
| 1-2 hours after meals / postprandial peak | <180 mg/dL / <10.0 mmol/L |

Targets vary by age, pregnancy status, medication, hypoglycemia risk, and clinician plan. Prefer the user's personal clinical target if provided.

## Visualization Bands

Use these default chart bands when no personal target is known:

- Low: <3.9 mmol/L / <70 mg/dL, red.
- Fasting/pre-meal expected band: 3.9-5.6 mmol/L / 70-99 mg/dL for general reference; 4.4-7.2 mmol/L / 80-130 mg/dL if the user is tracking diabetes-management targets.
- 2h post-meal general reference: <7.8 mmol/L / <140 mg/dL.
- Post-meal diabetes-management target: <10.0 mmol/L / <180 mg/dL.
